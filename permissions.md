##### Exemplo

`drwxr-xr-x`  

`d` = diretório  
`-` = arquivo comum de usuário  
`c` = arquivo de caractere  
`b` = arquivo de bloco  
`l` = link  

| Proprietário | Grupo | Outros |
| ----------------------------- |
| `rwx`        | `r-x` | `r-x`  |  

`r` Read  
`w` Write  
`x` Execution  
`-` No permission
